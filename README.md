#### Spirit Taekwondo Tracker
Allows user to track the classes they went to, they plan to go to and displays future classes.


### Technologies Used
* Javascript/Embedded JS
* Express/Node.js/RESTful API
* MongoDb/Mongoose
* Bootstrap
